Manage and display your portfolio projects using a customizable grid layout

== Description ==

Manage and display your portfolio projects using a customizable grid layout. This plugin is useful for any business that wants to show their projects to their website's visitors.

With **a-folio** you can easily:

* Manage (add / remove / edit) Portfolio items (projects) in wp-admin
* Insert the portfolio grid into any page using a shortcode
* Select between 5 different row layouts, create repeating patterns with them
* Add a CTA tile to your stream with a custom link
* Customize the tiles using the plugin's settings or with custom CSS classes

You can find detailed documentation on the plugin's website:
[https://a-idea.studio/a-folio](https://a-idea.studio/a-folio)

== Installation ==

1. Go to **Plugins >> Add New** in wp-admin
1. Search for "a-folio"
1. Click on the **Install Now** button
1. Then on **Activate**
1. After you installed and activated the plugin, a new submenu item will appear in the WP-admin area (You’ll be able to find it after installation in **Settings » a-folio Settings**)
1. You can manage Portfolio items in wp-admin's **Portfolio items** section

== Frequently asked questions ==

= How can I add Portfolio items (projects)? =

Just go to **Portfolio items » Add New**. It works in the same way as adding posts or pages.

= Where can I add the description to the tile? =

The Excerpt is used as the project description on the tiles. It means that if you don't specify the excerpt then the auto-generated excerpt (based on The Content) will be used.
